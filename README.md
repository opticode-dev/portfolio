## アプリ名
My Portfolio

## 概要
自作アプリが複数になり、今後も新たに自作していく予定のため、リンクをまとめて管理し、表示できるサイトを作りたいと考えました。
そこで、ポートフォリオサイトとして、自作アプリの他にも私自身に関する情報を記載することにしました。
随時情報を追加していきます。

## 使用技術
VSCode
Ubuntu
Linux
Python
Streamlit
GitHub

## 主な機能
ポートフォリオとして項目ごとに情報を記載しています。

## 工夫した点
見やすいように、ヘッダーごとに箇条書きで内容をまとめました。
Githubリンク、Streamlit Community Cloudにより作成したWEBアプリケーションのリンクをそれぞれ添付し、詳細を確認できるようにしました。

## 今後の改善点
HTML/CSSの学習も兼ねて、HTML/CSSの技術を利用してUI/UX等の改善をし、デザイン性の向上に努めます。
